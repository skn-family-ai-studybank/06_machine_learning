# 🔎 Machine Learning

## 머신러닝 용어 정리

> 머신러닝: 데이터에서 규칙을 학습하여 새 데이터에 대한 예측을 수행하는 것

### 기본 용어

| 용어 | 의미 |
| --- | --- |
| feature | 모델이 판단할 때 이용하는 입력 값 (1열) |
| target / label | 모델이 맞춰야 하는 정답 |
| `X` | 입력 데이터의 모음 |
| `y` | 입력 데이터의 정답 모음 |

### 학습 방식

- **지도 학습**: 정답이 있는 데이터를 모델에 학습시키는 방식
  - **분류**: 정답이 클래스인 문제 (예: 생선 종류 예측, 와인 종류 예측)
  - **회귀**: 정답이 연속적인 숫자인 문제 (예: 주택 가격 예측, 무게 예측)

### 전처리

- **스케일링**: feature 값(숫자)의 범위를 맞추는 전처리
  - 한 feature 값의 범위가 다른 feature보다 넓으면, 범위가 큰 feature가 모델의 판단을 지배하는 문제가 발생함
  - 이를 방지하기 위해 스케일링을 사용

### 기초 모델

- **KNN**: 새 데이터와 가까운 이웃 데이터를 이용해서 예측하는 모델
  - 분류: 가까운 이웃들의 다수결로 class 예측
  - 회귀: 가까운 이웃들의 평균값으로 예측

### 회귀 모델

- **선형 회귀**: 입력값과 정답 사이의 관계를 직선(선형식)으로 표현하는 모델
  - 기울기(가중치), 절편, 회귀계수
- **다중 선형 회귀**: feature가 여러 개인 선형 회귀
  - 여러 입력값을 함께 사용해서 target을 예측

### 과적합

- **과소적합**: 모델이 훈련 데이터에 너무 덜 맞아서 새 데이터에 대한 예측 성능이 떨어짐 (학습을 너무 적게 함)
- **과대적합**: 모델이 훈련 데이터에 너무 잘 맞아서 새 데이터에 대한 예측 성능이 떨어짐 (학습을 너무 많이 함)

### 규제 선형 모델

모델이 너무 복잡해지지 않도록 회귀계수에 벌점을 주는 방법

- **L1 (Lasso)**: 일부 계수를 0으로 만듦
- **L2 (Ridge)**: 계수를 작게 제한하도록 벌점 부여
- **L1 + L2 (Elastic Net)**: Lasso와 Ridge를 함께 적용

### 분류 모델

- **로지스틱 회귀**
  - 이름만 회귀고 실제로는 분류 모델
  - 선형 방정식의 결과를 sigmoid 또는 softmax 함수를 이용해서 클래스를 예측하는 모델

### 트리 / SVM

- **결정 트리**
  - 데이터를 feature 조건으로 나누면서 예측하는 모델
  - 분류: 조건에 맞춰 분할된 리프 노드 sample의 다수결로 class를 예측
  - 회귀: 조건에 맞춰 분할된 리프 노드 sample의 평균값으로 예측
  - **가지치기**: 결정 트리가 복잡해지는 것을 막음 (과대적합 방지)
    - `max_depth`, `min_samples_leaf`, `min_samples_split` 사용

- **SVM**
  - class를 나누는 결정 경계를 찾는 모델
  - 핵심: 마진이 넓은 결정 경계를 갖는 것
  - `C`, `kernel`, `gamma` 파라미터 값을 이용해서 조정

### 앙상블

여러 모델을 함께 사용해 더 안정적인 예측을 하는 기법 (단일 모델보다 보통 성능이 좋아짐)

- **Voting (투표)**: 여러 모델의 예측을 투표로 합침
  - Hard: class 예측값으로 투표
  - Soft: class 확률을 평균내어 예측

- **Bagging**
  - 같은 모델을 여러 개 만들고 결과를 합치는 방식
  - 각 모델은 bootstrap sampling 방식으로 데이터를 추출해 학습

- **Random Forest**
  - Bagging을 결정 트리에 적용한 대표 모델
  - 데이터와 feature를 랜덤으로 선택해서 여러 트리를 학습
  - feature importance로 모델이 중요하게 본 feature 확인 가능

- **Boosting**
  - 이전 모델의 틀린 부분(오차)을 다음 모델이 보완하는 방식
  - 여러 약한 모델을 순차적으로 학습시켜 강한 모델을 만듦

- **AdaBoost**
  - 틀린 샘플의 가중치를 높여 다음 모델이 더 집중하게 만듦
  - 학습이 마무리되면 모델(트리)별 신뢰도 가중치를 부여

- **Gradient Boosting**
  - 이전 모델의 오차를 다음 모델이 줄이는 방식
  - 개선한 라이브러리: XGBoost, CatBoost, LightGBM

- **Stacking**
  - 여러 기본 모델의 예측 결과를 모아서 최종 모델이 다시 학습
